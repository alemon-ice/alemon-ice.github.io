Apresentação

Seja bem-vindo à minha página introdutória e meu primeiro projeto.

Estou iniciando no desenvolvimento web e esta página é um projeto que criei como resultado de um desafio que propus à mim mesmo.

Detalhes do Desafio

O desafio consistiu em montar um cronograma de estudos envolvendo as tecnologias web HTML, CSS e JavaScript e trabalhar com a integração dessas três tecnologias durante um período de 3 semanas.

Após finalizar o desafio, nada melhor do que praticar para fixar o conteúdo na mente, pensando nisso foi que decidi construir essa página, por mais simples que seja.

O conteúdo abordado foi introdutório sobre o uso puro dessas tecnologias e ainda há muito o que aprender sobre desenvolvimento web e minha jornada está apenas começando.

Portanto continuarei buscando melhorar e utilizarei este blog para registrar meu progresso.

Agradeço pela atenção!
